# Звездный десант "17

Аркадный вертикальный шутер с боями против волн врагов и босса.

## Управление на ПК

- `W`, `A`, `S`, `D` или стрелки: движение
- `Space`: стрельба
- `Esc`: пауза в игре, выход назад из экранов меню
- `Enter` / `Space`: подтверждение в меню

## Режим разработчика

Режим разработчика включается в меню `Настройки`.

Во время игры доступны отладочные команды:

- `Z`: godmode
- `B`: overlay
- `V` / `C`: таймер этапов
- `Q` / `E`: скорость игры

## Актуальные UI-фиксы

- Меню и overlay-экраны адаптированы под малую высоту viewport.
- Для экранов меню добавлено безопасное вертикальное масштабирование без выхода элементов за верхнюю и нижнюю границы.
- Предупреждение перед боссом исправлено для корректного отображения кириллицы.

## Локальный запуск

Требуется Python 3.

```powershell
py -3 -m http.server 3000 --bind 127.0.0.1
```

После запуска откройте в браузере:

```text
http://127.0.0.1:3000/
```
