# RUN ANALYTICS RUNTIME

Назначение:
- зафиксировать минимальный набор runtime-данных, которые игра должна собирать во время забега
- стать единым источником данных для результатов, комментариев и рейтинга
- без реализации, без формата хранения, без кода

## Минимальный набор данных

- итог забега: победа / поражение / принудительное завершение
- итоговые очки
- время прохождения
- финальный ранг
- факт нового рекорда
- стадия, на которой завершился забег
- дошёл ли игрок до босса
- был ли босс уничтожен
- сколько здоровья осталось у босса на момент завершения
- общее число уничтоженных врагов
- уничтожения по типам врагов
- очки по источникам
- полученный урон по источникам
- основная причина смерти
- топ-3 источника полученного урона
- был ли забег лучше предыдущего по очкам
- был ли забег лучше предыдущего по рангу

## Очки по источникам

- `Basic`
- `Tank`
- `Zigzag`
- `Web`
- ноги босса
- тело босса
- бонусы за забег, если они будут добавлены позже

## Урон по источникам

- ракета танка
- паутина
- лазер босса
- spread-атака босса
- столкновения
- прочие будущие источники

## Дополнительные флаги

- почти убил босса
- победил босса с запасом
- проиграл на боссе
- дошёл до босса без победы
- ранняя смерть
